areSubstrings
Default value : ( str1: Optional<string>, str2: Optional<string> ): boolean => { if (!str1 || !str2) { return false; } const normalizedStr1 = normalizeForComparison(str1); const normalizedStr2 = normalizeForComparison(str2); return ( normalizedStr1.includes(normalizedStr2) || normalizedStr2.includes(normalizedStr1) ); }

Checks if one of the strings is contained in the other. The comparison is made on the normalized value of each string.
normalizeForComparison
Default value : (text: string): string => text .toLowerCase() .normalize("NFD") .replace(/\p{Diacritic}/gu, "")

Sets a text to lowercase and removes diacritics.
